Banking Management System
This is a simple Java-based banking application that allows users to view their account balance and transfer money between accounts.
It demonstrates the use of Java classes, exception handling, and object-oriented principles.
 
**Features**
View Balance: Check the balance of a specified account.
Transfer Money: Transfer funds from one account to another.
Exception Handling: Handles scenarios such as insufficient funds during a transfer.
